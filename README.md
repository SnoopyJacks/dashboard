# Dashboard UI (Static)

A small multi-page admin-style dashboard built with **HTML + CSS** (no frameworks).  
Includes: sidebar navigation, KPI cards, simple SVG/HTML charts, and a table.

## Pages
- `index.html` — overview
- `reports.html` — reports chart
- `analytics.html` — analytics KPIs
- `settings.html` — settings

## Run locally
Just open `index.html` in your browser.

## Notes
- This is a UI demo (static data).
- Responsive layout added for smaller screens.
